# Multithreading OpenCV in C++
An implementation of multithreading on OpenCV to improve the frame per second (FPS) rate 

## Instruction
Build and execute:
```shell
mkdir build
cd build
cmake ../
make
```

Remove files in .gitignore:
```shell
chmod 700 clean.bash
./clean.bash < .gitignore
```
